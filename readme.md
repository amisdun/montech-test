### You would need:

- Node.js
- MongoDB

### How to run the project:

```
  yarn or npm install
  yarn run start
  yarn run test
```

`Create a .env file at the root of the folder and copy content from .env.example to .env`

##### Define the following environment variables as follows

```
    MONGODB_URI=
    PORT=
    JWT_SECRET=
    TMDB_API_KEY=
    TMDB_API_URL=
```
